## Como baixar e iniciar o projeto

```bash
# Clonar o repositório
$ git clone https://github.com/thimfont/hackday-codehints.git

# Entrar no diretório
$ cd hackday-codehints

# Instalar as dependências
$ yarn install 

# Iniciar o projeto
$ yarn start

```
